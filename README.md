# GitHub User Finder

**Автор:** Коваленко Кирилл

## Описание
Простое GUI-приложение для поиска пользователей GitHub, с возможностью добавлять их в избранное и сохранять этот список в JSON-файл.

## Использование API
- Введите имя пользователя в поле поиска и нажмите "Поиск".
- Двойным щелчком по результату добавьте пользователя в избранное.
- Нажмите "Показать избранных" для просмотра сохранённых пользователей.

## Примеры тестов
- Поиск существующего пользователя (`torvalds`)
- Проверка добавления нескольких пользователей
- Проверка сохранения и загрузки из JSON
